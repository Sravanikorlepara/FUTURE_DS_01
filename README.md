# 📊 Business Sales Performance Analytics

## 📌 Project Overview

This project was completed as part of the **Future Interns – Data Science & Analytics Virtual Internship (Task 1)**.

The objective of this project is to analyze business sales performance using the Superstore Sales dataset and build an interactive Power BI dashboard that provides meaningful business insights for decision-making.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Track revenue and profit trends over time.
- Compare sales across different regions.
- Identify top-selling products.
- Analyze category-wise sales and profit.
- Generate actionable business recommendations.

---

## 🛠️ Tools Used

- Microsoft Power BI
- Microsoft Excel
- CSV Dataset
- GitHub

---

## 📂 Dataset

- **Dataset:** Superstore Sales Dataset
- **Source:** Kaggle
- **Records:** Approximately 9,994

---

## 📈 Dashboard Features

- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Average Sales KPI
- Sales Trend Analysis
- Sales by Region
- Sales by Category
- Profit by Category
- Top 10 Products by Sales
- Interactive Slicers

---

## 💡 Key Insights

- Technology products generated the highest profit.
- The West region recorded strong sales performance.
- Sales showed seasonal fluctuations.
- A small number of products contributed significantly to revenue.
- Furniture had relatively lower profitability.

---

## 🚀 Business Recommendations

- Increase inventory for top-selling products.
- Improve pricing strategies for low-profit categories.
- Expand marketing efforts in high-performing regions.
- Plan promotional campaigns during peak sales seasons.
- Use dashboards for continuous business monitoring.
